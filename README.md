 
# Book AI Assistant - Frontend

Frontend del chatbot académico para buscar información en el libro de IA.

## Características
- Interface conversacional moderna
- Búsqueda semántica en tiempo real
- Resultados con porcentaje de relevancia
- Copiado de textos al portapapeles
- Diseño responsivo

## Tecnologías
- Next.js 15 con App Router
- TypeScript
- Tailwind CSS
- Axios para API calls
- Lucide React para íconos
